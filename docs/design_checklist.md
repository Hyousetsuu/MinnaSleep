# Neo Brutalism Design QA Checklist

- [ ] All buttons use hard shadows.
- [ ] Border width is minimum 3px.
- [ ] Design Tokens are strictly used across all components.
- [ ] NO hardcoded colors.
- [ ] Typography strictly follows `AppTypography`.
- [ ] Minimum touch target size is 48dp.
- [ ] Contrast meets WCAG standards.
- [ ] Dark & Light Theme remains visually consistent.
- [ ] Animations follow `AppAnimations` (Fast/Normal/Slow/Hero).
- [ ] Icons strictly use `AppIcons` (Lucide Icons).
