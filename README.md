# 🌙 Minna Sleep Platform

A full-stack sleep tracking platform built with:
- **Mobile App**: Flutter, Riverpod, Drift (Offline-first)
- **Backend API**: Express.js, Supabase, Midtrans, Gemini AI
- **Admin Dashboard**: React (Future)

## Architecture
See `shared-docs/architecture.md`

## Setup
### Backend
```bash
cd backend-api
npm install
npm run dev
```

### Mobile App
```bash
cd mobile-app
flutter pub get
flutter run
```
